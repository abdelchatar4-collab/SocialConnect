# SocialConnect

**SocialConnect** est une application web moderne conçue pour faciliter et optimiser la gestion des usagers dans le secteur social et les services publics. Développée avec les dernières technologies web (Next.js), elle offre une interface intuitive et performante pour le suivi des dossiers, la gestion des interventions et l'analyse des données sociales.

## Fonctionnalités Principales

- **Gestion centralisée des usagers** : Création, modification et suivi des dossiers usagers.
- **Suivi des dossiers** : Historique des interactions, notes et documents associés.
- **Analyse et Reporting** : Outils visuels pour l'analyse des données (prévention expulsion, logement, etc.).
- **Interface Ergonomique** : Design soigné et navigation fluide pour une meilleure expérience utilisateur.
- **Sécurité et Confidentialité** : Respect des normes de protection des données.

## Installation et Utilisation

### Prérequis

- Node.js (version 18 ou supérieure recommandée)
- npm, yarn, pnpm, ou bun

### Démarrage Rapide

1.  **Cloner le projet** (si nécessaire) ou accéder au dossier du projet.
2.  **Installer les dépendances** :
    ```bash
    npm install
    # ou
    yarn install
    ```
3.  **Lancer le serveur de développement** :
    ```bash
    npm run dev
    # ou
    yarn dev
    ```
4.  **Accéder à l'application** : Ouvrez votre navigateur sur [http://localhost:3000](http://localhost:3000).

Pour plus de détails sur le déploiement ou les scripts disponibles, consultez les fichiers de configuration (`package.json`).
